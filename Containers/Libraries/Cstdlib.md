# Cstdlib

### abs

Returns the absolute value of parameter |n|

```cpp
int abs (int n);
cout<<abs(-3);
```

### llabs

Returns the absolute value of parameter |n|

```cpp
long long int llabs (long long int n);
cout<<llabs(-1e10);
```

### fabs

Returns the absolute value of parameter |n|

```cpp
double fabs (double x);
cout<<fabs(-3.987);
```
